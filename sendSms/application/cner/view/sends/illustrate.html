短信接口说明

入参列表(POST、GET)
参数名 参数类型 必填与否 样例取值 参数说明
num String 必填 13000000000 短信接收号码。支持以逗号分隔的形式进行批量调用，批量上限为1000个手机号码,批量调用相对于单条调用及时性稍有延迟,验证码类型的短信推荐使用单条调用的方式
key String 必填 uynDKQ6pbNrCdtB 后台生成的Key
url String 必填 cnereate.xyz key对应的网站url
appid String 必填 cdxy Key对应的AppId
token String 必填 sdlwe1kl3j3i4i33k2l 采用拼接appid+appid的值+time+time的值+key+key的值成为字符串，把拼接后的字符串进行先MD5后sha1的加密方式生成
time String 必填 1500000000 当前时间戳
sign String 必填 信儿众创 短信签名
TemplateCode String 必填 SMS_110280059 模板code
param String 必填 {"num":"'18281602752'，验证码'674664'"} 短信模板变量替换JSON串,友情提示:如果JSON中需要带换行符,请参照标准的JSON协议对换行符的要求,比如短信内容中包含\r\n的情况在JSON中需要表示成\r\n,否则会导致JSON在服务端解析失败
出参列表
参数名 参数类型 样例取值 参数说明
Message String OK 状态码的描述
code String {"num":"'18281602752'，验证码'674664'"} 短信模板变量替换JSON串
